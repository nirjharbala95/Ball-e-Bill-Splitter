# Ball-e's Bill Splitter

A web-based bill splitting utility that helps you fairly divide expenses among a group of people. Simply add items, assign participants, and let the app calculate each person's share—including proportional tax distribution.

## Features

- **Add Bill Items** - Enter item name, price, and participants (comma-separated names)
- **Edit & Delete** - Modify or remove items as needed
- **Tax Support** - Add a flat tax amount that gets distributed proportionally
- **Per-Person Breakdown** - See each person's subtotal, tax share, and total owed
- **Real-time Calculations** - Hit Calculate to see the complete breakdown

## Tech Stack

- Vanilla HTML, CSS, and JavaScript
- No frameworks or build tools required
- Works in any modern browser

## Usage

1. Open `index.html` in your browser
2. Enter an item name, price, and participants (comma-separated)
3. Click "Add Item" to add it to the bill
4. Repeat for all items
5. (Optional) Enter a tax amount
6. Click "Calculate" to see each person's share
